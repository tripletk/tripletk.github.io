ThinDan (YGN Tuitions)
Angel Hack Yangon 2018

ThinDan is an application that allows students to locate tutors with ease within the city of Yangon, and allows aspiring educators to meet with their future students. It combines fast app logic with an intuitive user interface for maximal efficiency. It is programmed using the latest technologies; a frontend and backend written in Swift and a fast server side in Go as well as a Firebase implementation in Swift.
